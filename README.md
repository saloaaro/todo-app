# Todo-App 📝

A simple and modern **Todo List web application** built with **HTML, CSS, and JavaScript**.  
This app allows users to add, complete, and delete tasks, with data saved in the browser's **localStorage**.

---

## Demo

Check the live version here: [Todo-App Live](https://saloaaro.github.io/todo-app/)

---

## Features

- Add new tasks quickly using the input field.  
- Mark tasks as completed.  
- Delete tasks easily.  
- Tasks are **saved in localStorage**, so they persist across page reloads.  
- Clean, modern UI with responsive design.  
- Hover effects and visual feedback for a better user experience.

---

## Technologies Used

- **HTML5** – Markup structure  
- **CSS3** – Styling, hover effects, responsive layout  
- **JavaScript** – Task management, localStorage
